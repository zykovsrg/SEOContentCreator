## Текущий changelog

### 2026-05-23 — под-проект 5 «Редактор промтов этапов» + багфикс GPT-5

- Change: Завершён под-проект 5 (срез A раздела «Шаблоны»): единый источник дефолтов StageTemplateDefaults, справочник переменных TemplateVariables, TemplatesView (список этапов + редактор системного/пользовательского промта + параметры модели + «Сбросить к стандартному»). Дополнительно (прошлая сессия): поддержка GPT-5/o-series (max_completion_tokens), предупреждение об обрыве по лимиту токенов, расширенный список моделей. Багфикс этой сессии (коммит 2f0f624): для GPT-5/o-series не отправляем temperature (OpenAI её запрещает — раньше «Черновик» молча не генерировался); красная плашка ошибки в рабочем пространстве; человекочитаемые тексты ошибок (OpenAIError → LocalizedError).
- Impact: Редактор промтов этапов рабочий; генерация на GPT-5-совместимых моделях больше не падает из-за температуры; ошибки видны пользователю. Следующий — под-проект 6 (по согласованию: срезы «Шаблонов» B–E, этап «Стиль», полная «Семантика», очередь, публикация в Google Docs).
- Manual checks: Компиляция (приложение + тесты) зелёная (build-for-testing). Smoke (пользователь): генерация работает (gpt-4.1), правка промта влияет на результат, «Сбросить к стандартному», параметры модели, справочник переменных — подтверждено; плашка ошибки показала читаемый текст (401 при неверном ключе, 404 для недоступной GPT-5-модели). Юнит-тесты из CLI не выполнялись (тест-раннер зависает — особенность среды); запуск — Cmd+U.

### 2026-05-22 — под-проект 4 «Проверяющие этапы»

- Change: Реализован под-проект 4 «Проверяющие этапы» (TDD, 11 задач): 3 проверяющих этапа (Проверка SEO, Фактчекинг, Финальная вычитка) с агентами ИИ-SEO/фактчекер/редактор; замечания в JSON (Remark + RemarksParser), точечное применение (RemarkApplier), ветка проверок в StageExecutor (transient remarks, без версии), переменная {{база_знаний}} для фактчекера, 3 стартовых шаблона с методичками; UI — панель карточек замечаний с подсветкой фрагмента и точечным приёмом, «Готово» создаёт одну версию «Правки проверки».
- Impact: Пайплайн статьи дополнен контролем качества (SEO, факты, язык) — ИИ предлагает правки, человек принимает точечно. Следующий — под-проект 5 (кандидаты: «Шаблоны», «Стиль», полная «Семантика», очередь, публикация).
- Manual checks: Все unit-тесты зелёные. Smoke (Cmd+R): «Финальная вычитка» вернула 9 замечаний по категориям, подсветка фрагмента, приём правок вживую, «Готово» → версия — пользователь подтвердил. По фидбэку увеличен шрифт карточек.
